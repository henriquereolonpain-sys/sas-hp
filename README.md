# HP_ — Landing de Serviços Financeiros

Landing page de serviços financeiros de **Henrique Pain**: DRE gerencial, fluxo de caixa, projeções & cenários, análise de dados/dashboards e consultoria financeira para PMEs.

**Stack**: HTML + CSS + JavaScript puros, sem build. Fontes Google (Space Grotesk + Inter), imagens otimizadas em WebP.

## Estrutura

- `index.html` / `style.css` / `script.js` — versão atual (dark fintech, accent verde-lima)
- `v1/` — primeira versão, estética sóbria derivada do [portfólio](https://henriquereolonpain-sys.github.io)
- `img/` — foto de perfil e backgrounds (Unsplash, licença livre)

## Rodar localmente

```bash
python -m http.server 8123
# http://localhost:8123
```
